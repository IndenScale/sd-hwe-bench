What is SD-HWE-Bench?

SD-HWE-Bench evaluates AI agents on declarative engineering design. Given a natural-language engineering requirement, an agent must produce structured YAML design declarations that pass automated rule checks (L0–L4) and generate valid engineering deliverables — BOM, port maps, rack panel diagrams, and power budgets.

Why HWE needs its own SWE-Bench

Software engineering RLVR works because code has compilers, linters, and test suites — fast, deterministic, interpretable verifiers. Hardware engineering has even richer verifiable constraints, but lacks the infrastructure to expose them to AI agents. SD-HWE-Bench provides that infrastructure, making hardware engineering capability measurable, competitive, and reproducible.
