# ESA 规则目录

截至 2026-06-20，piki 共实现 64 条规则，其中 62 条由插件注册，2 条为内置接口检查。这些规则主要分布在 ESA 核心层级 L2–L3，部分快速几何规则属于 L4a 扩展；L0–L1 的解析与 schema 校验由 ADL 加载器执行，不归类为 ESA 规则。

## 电信插件规则示例

电信插件是规则最密集的插件，其规则覆盖从接口兼容到空间布局的多个维度。以下为部分代表性规则：

| 规则 ID | 名称 | 层级 | 说明 |
|---|---|---|---|
| `INTERFACE-COMPAT-001` | 接口类型兼容性检查 | L2 | 确保 Mate 或 Connection 两端的接口类型可配对 |
| `INTERFACE-CABLE-001` | 线缆类型与接口匹配检查 | L2 | 确保所选 cable_type 与端口接口类型一致 |
| `TELECOM-POWER-001` | PDU 功率预算检查 | L3 | 校验 PDU 负载不超过容量阈值 |
| `TELECOM-RACK-001` | U 位冲突检查 | L3 | 校验同一机柜内设备 U 位不重叠 |
| `TELECOM-RACK-002` | 机柜容量检查 | L3 | 校验设备总高度不超过机柜可用 U 位 |
| `TELECOM-COLLISION-001` | 机柜内设备 3D 碰撞检测 | L4a | 使用 AABB 快速几何校验设备空间冲突 |
| `TELECOM-CONN-002` | 连接端口类型兼容性检查 | L2 | 校验 Connection 两端端口类型一致 |
| `TELECOM-CONN-003` | 连接线缆类型匹配检查 | L2 | 校验 cable_type 与端口速率匹配 |
| `TELECOM-WEIGHT-001` | 机柜承重检查 | L3 | 校验机柜内设备总重量不超过承重限制 |
| `TELECOM-FLOOR-002` | 机柜维护通道宽度检查 | L4a | 校验同排机柜间距满足维护通道要求 |

完整规则实现位于 `src/piki/extensions/<plugin>/plugin.py`，规则元数据（ID、名称、优先级、严重级别）通过 `@rule(rule_id, name, priority, severity)` 装饰器注册。

## 规则与 ESA 层级的对应

- **L2 规则**：接口兼容性、Mate 自洽性、引用存在性。这类规则在 ADL 加载后即可执行，不依赖复杂项目级上下文。
- **L3 规则**：跨记录业务规则，如功率预算、承重、容量。需要遍历多个集合（如所有设备、所有 PDU）并聚合计算。
- **L4a 规则**：快速几何规则，如 U 位冲突、AABB 碰撞、维护通道宽度。依赖布局解析后的几何信息，但避免调用重型 CAD 内核。

规则的分层注册使得项目可以按需启用规则包：例如，纯逻辑设计阶段可只启用 L2 规则，进入详细布局与空间规划阶段后再启用 L3–L4a 规则。L0–L1 的解析与 schema 校验始终启用，作为所有后续检查的前置条件。
