# References

This directory stores local copies of papers used while drafting the Engineering as Code evaluation substrate paper.

## HWE-Bench: Real-world hardware bug repair

- Local file: `hwe-bench-rtl-bug-repair-2604.14709.pdf`
- arXiv: <https://arxiv.org/abs/2604.14709>
- Role in this paper: repository-level RTL/Chisel bug repair shows that hardware-agent benchmarks work when a domain already has code-like representation, executable simulation/regression flows, and tool-consumable hardware knowledge.

## HWE-Bench: Board-level schematic design

- Local file: `hwe-bench-board-schematic-2603.18102.pdf`
- arXiv: <https://arxiv.org/abs/2603.18102>
- Role in this paper: board-level schematic generation shows another favorable hardware domain where datasheets, schematic/netlist representations, electrical rules, and circuit simulation instantiate an evaluation loop.
