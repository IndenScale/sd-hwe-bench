# AIDC 施工与可建造性规范

## 1. 吊装点

- 每台重大设备（冷水机组、变压器）必须在 `construction/hoisting-plan.yaml` 中声明独立吊装点。
- 吊装点需包含：设备 ID、吊装点坐标（x_mm, y_mm, z_mm）、吊车吨位（ton）、回转半径（m）、与最近障碍物的净距（mm）。
- 单台冷水机组重量约 15t，变压器约 20t；主吊车额定起重量不得低于 1.25 倍设备重量。
- 吊装路径不得穿过车辆通道或机柜维护通道。

## 2. 施工机具租赁

- `construction/equipment-rental.yaml` 需列明：主吊车、叉车、临时发电车、高空作业平台的租用起止日期与费用。
- 租赁周期应覆盖对应设备的吊装与就位窗口。

## 3. VDC 工作面

- `construction/vdc-workface.yaml` 按施工阶段划分工作面：土建移交、机电安装、设备就位、调试。
- 每个工作面需声明：使用区域、进场门、临时堆场面积、允许作业风速（m/s）、与运行中设备的隔离要求。

## 4. 几何约束

- 机柜与冷水机组/变压器之间保留 ≥2000mm 维护通道。
- 门开合扫掠区内不得有机柜或大型设备。
- 主物流通道宽度 ≥4000mm。
