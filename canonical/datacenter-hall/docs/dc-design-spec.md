# AIDC 主机房设计规范

## 1. 机房物理参数

| 参数 | 值 |
|------|-----|
| 尺寸 (L×W×H) | 12m × 8m × 3.2m |
| 面积 | 96 m² |
| 体积 | 307.2 m³ |
| 设计室内温度 | 24°C ± 2°C |
| 设计相对湿度 | 45% ± 10% |
| 围护结构传热系数 | 0.5 W/(m²·K) |

## 2. 机柜与设备

- 2 排 × 4 机柜 = 8 台 42U 机柜
- 每机柜额定功率 10kW，冷却容量 8kW
- 每机柜部署：2 台 2U 计算节点 (TDP 500W) + 1 台 2U 存储节点 (TDP 350W)
- A01/A02 机柜各额外部署 1 台 4U GPU 节点 (TDP 2000W)
- 总计：16 计算节点 + 8 存储节点 + 2 GPU 节点

## 3. 冷却系统

- 每排 2 台列间精密空调，单台制冷量 30kW
- 总额定制冷量 120kW
- COP 随负载率变化，名义 COP 4.5
- 当室外温度 ≤ 15°C 时可启用免费冷却（Free Cooling）

## 4. 配电系统

- 市电容量 200kW
- 屋顶光伏 20kWp，逆变器效率 97%
- 储能柜 50kWh，充放电效率 92%，最大充放电功率 25kW
- 每机柜双路 PDU（A/B 路），单路容量 22.08kW
- 电网碳排放因子：0.581 kgCO₂/kWh（中国平均）

## 5. 运营约束

- PUE 目标：≤ 1.4
- 室内温度不得持续超过 28°C
- 冷却系统最小部分负载率 20%
- 储能 SOC 不得低于 10%（保护电池寿命）
- 市电中断时储能需支撑满负荷 15 分钟以上
