# AIDC 60MW 主机房设计规范

## 1. 项目概述

本项目为 60MW AI 数据中心（AIDC）主机房高精度 ADL 模型，用于 SD-HWE-Bench 的 AIDC 运营优化与设计-调度联合优化任务。

- **机房规模**：80m × 40m × 5.5m，4 排 × 50 机柜 = 200 台 42U 机柜
- **IT 负载**：1000 台 8U 液冷 AI 训练服务器，单台 TDP 60kW，总 TDP 60MW
- **单机柜功率密度**：300kW（液冷配置）
- **冷却架构**：液冷为主（中央冷水机组 + CDU + 冷板），预留风冷（CRAH）接口
- **配电**：中压 110/10kV，双路 40MVA 变压器 N+1

## 3. 冷却系统

- **8 台 10MW 离心式冷水机组**，总额定制冷 80MW（N+1 冗余）
- **名义 COP**：4.5（高温夏季工况）
- **COP 曲线**：`[[0.2, 3.0], [0.4, 3.8], [0.6, 4.5], [0.8, 4.8], [1.0, 4.5]]`
- **免费冷却阈值**：湿球温度 ≤ 24°C
- **免费冷却条件**：冷冻水设定点 ≥ 16°C 且湿球温度 ≤ 24°C
- **冷却塔耗水**：1.2 L/kWh_th

## 6. 运营策略设计空间

Agent 可调决策变量：

- `it_utilization`：每小时 IT 负载率 [0, 1]
- `chiller_setpoint_c`：冷冻水温度设定点 [8, 26]
- `battery_mode`：`charge` | `discharge` | `idle` | `auto`

优化目标：

- PUE（30%）
- 碳排放（30%）
- 水消耗（15%）
- 电费（25%）

## 7. 关键提示

1. 夜间湿球温度低时，提高冷冻水设定点可触发免费冷却，显著降低冷却功耗。
2. 设定点越高，冷水机组 COP 越高，但供冷能力下降；需保证室内温度不超过 32°C。
3. 储能可在谷电/光伏余电时充电，峰电时放电，降低电费。
4. 变压器效率随负载率变化，高负载率时效率更高。
